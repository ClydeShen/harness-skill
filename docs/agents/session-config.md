# Session Config

**State file:** `.planning/STATE.md`
**GSD-compatible:** yes

## Phases
| Phase | Description |
|---|---|
| `01-discuss` | Explore and define scope |
| `02-plan` | Write specs and tasks |
| `03-execute` | Implement |
| `04-verify` | Confirm and ship |

## Session Start
Run `/session-start` to resume from last session state.

## Session Handoff
Run `/context-handover` to package context for the next session.
