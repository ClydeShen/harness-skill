# Triage Labels

## Core Labels
| Label | Meaning |
|---|---|
| `status:needs-triage` | Newly filed, not yet reviewed |
| `status:needs-info` | Blocked on clarification from author |
| `status:ready-for-agent` | Scoped and ready for AI work |
| `status:ready-for-human` | Needs a human decision |
| `status:wontfix` | Closed, out of scope |

## Extended Labels
| Label | Meaning |
|---|---|
| `status:needs-prd` | Needs a product requirements doc before work begins |
| `status:needs-review` | Awaiting code or design review |
| `status:in-progress` | Actively being worked |
| `status:done` | Completed and verified |
